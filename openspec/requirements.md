# Project Requirements

**Stack:** java-spring-boot, angular, web-design-guidelines

## General Context (About)
Plataforma web integral para la gestión de alquiler de vehículos. El sistema permite a los clientes explorar un catálogo interactivo de autos, visualizar especificaciones técnicas (capacidad, categoría, precio por día), y gestionar sus reservas. El objetivo es ofrecer una experiencia de usuario moderna y fluida, apoyada por una arquitectura backend robusta que maneje el inventario, los estados de los vehículos y los pagos.
